PROJECTS

"Mastermind" in Ruby

Completed 31 Jan 2021 GitHub | Playable Replit

A version of the classic game "Mastermind", created in Ruby and playable in the console. The implementation allows for any combination of human and AI players. The AI applies an algorithm to select the optimal guess each turn, with a worst-case outcome of five guesses to arrive at the correct code.

To-do List Website

Completed 2 December 2020 GitHub | Live site

A website that allows users to create to-do lists and store them using localstorage. This is the most complex JavaScript application I have created to date. The visual style of the site is very barebones, as my focus was on improving with JavaScript.

In building this project, I gained familiarity with a number of new technologies and libraries, including JQuery, the "PubSub" model using the "pubsub-js" library, date manipulation using the "js-datepicker" and "date-fns" libraries, and more advanced uses of NPM and webpack. I also learned a lot about code organisation and building scaleable, maintainable projects.

Tic Tac Toe

Completed 11 October 2020 GitHub | Live site

A web implementation of Tic Tac Toe, built using JavaScript. The game can be played by any combination of human and AI players. The AI uses a "minimax" algorithm to select the optimal move, resulting in perfect play. Player scores are tracked across rounds.

This was one of the earlier projects I created using JavaScript and helped me to develop my fundamental knowledge of the language as well as practise algorithmic problem-solving and the use of closures and factory functions.

Calculator

Completed 20 September 2020 GitHub | Live site

A simple calculator application built using JavaScript. The calculator can perform basic mathematical operations and can be operated by clicking or using the keyboard.
